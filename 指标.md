1. ahr999 指标：（比特币价格/200日定投成本）*（比特币价格/指数增长估值）

   1. 前者可以看作上一批长期定投者的平均持仓成本
   2. 指数增长估值--数据模型对价格的估计拟合--视作内在价值增长线
2. MACD 指标：Moving Average Convergence Divergence指数平滑异同移动平均线

   1. DIF 线（快线）： 这是 MACD 的核心。它是用一条周期短的指数移动平均线减去一条周期长的指数移动平均线得到的。通常为 12日指数移动平均线 - 26日指数移动平均线。指代短期和长期趋势的差价，当 DIF 为正值，说明短期趋势强于长期趋势（看涨）；为负值则相反（看跌）。
   2. DEA线（慢线）：通常为 DIF 线的 9日指数移动平均线。DEA 线是对 DIF 线的平滑处理，它相对更稳定，滞后性更大。它的作用是产生交易信号。
   3. MACD 柱状图： 这是 DIF 线和 DEA 线的差值。(DIF - DEA) * 2，乘以2是为了让柱状图看起来更明显。柱状图的长短和正负反映了快慢线之间的离合关系，代表了趋势的加速度。柱子变长，说明趋势动能增强；柱子变短，说明动能减弱。
3. ATR指标：Average True Range，平均真实波动幅度。衡量资产价格的波动性大小

   1. 可以根据整个市场波动值，可以设置价格止损阈值
4. KDJ 指标：随机指数，包含K线、D线和J线。适用于短期振荡器，评估市场是处于“超买”还是“超卖”状态

   1. K线，市场的即时动态
      1. $RSV = \cfrac{今日收盘价 - 最近N天的最低价}{最近N天的最高价 - 最近N天的最低价} \times 100$
      2. RSV 的值在 0 到 100 之间。如果 RSV = 100，表示当日收盘价就是 9 天内的最高价，说明买盘极其强劲；如果 RSV = 0，则表示收盘价是 9 天内的最低价，卖盘极其强劲
      3. K值=（2/3） * 昨日K值 + （1/3） * 当日RSV。 这个公式一般代表指数平滑，即K值是RSV的平滑
   2. D线，K线的平滑版本，提供更稳定的市场趋势
      1. D值=（2/3） * 昨日D值 + （1/3） * 当日K值。即D值是K值的平滑
   3. J线，K 线与 D 线的乖离程度，旨在领先发出信号，波动幅度最大
      1. J 值 = 3 * 当日K值 - 2 * 当日D值
      2. 当日RSV+昨日K值-2* 昨日D值
5. RSI指标：Relative Strength Index，相对强弱指数。衡量近期价格变动的速度和幅度，以此评估市场是处于“超买”还是“超卖”状 态。

   1. RS = （N日内收盘价上涨幅度的平均值） / （N日内收盘价下跌幅度的平均值）
   2. RSI = 100 - [100 / (1 + RS)]
   3. 比较“特定时期内上涨收盘价的总和”与“下跌收盘价的总和”的相对大小。如果过去14天里，每天涨的都比跌的多，那么RS值就会很大，RSI就会靠近100。如果涨跌互现，但涨的多一点，RSI就会在50以上。如果每天跌的都比涨的多，RS值就会很小，RSI就会靠近0。
   4. 振荡器的统一缺点，钝化
6. 布林线指标：Bollinger Bands：随着价格波动而动态地扩张和收缩。它的核心作用是：衡量价格的相对高低、识别市场的波动性变化、以及捕捉潜在的趋势启动点。基于移动平均线，所以信号有一定滞后性。

   1. 中轨： 通常为 20期简单移动平均线。 代表中期趋势的基准线。
   2. 上轨： 中轨 + 2倍的标准差。 通常是价格运行通道的上方压力线。
   3. 下轨：中轨 - 2倍的标准差。通常是价格运行通道的下方支撑线。
   4. 它衡量的是价格相对于20期均线的平均波动范围。“2倍标准差”意味着，在正态分布下，约有95%的价格数据会落在这个通道之内
7. ROC指标：Rate of Change，变动率指标

   1. （当期收盘价 - N期前的收盘价） / N期前的收盘价] * 100；
      1. N 是设定的时间周期，常见的有 12天 或 25天。
      2. ROC > 0： 当期收盘价高于 N 期前的收盘价，市场处于上升动量中。
      3. ROC = 0： 当期收盘价等于 N 期前的收盘价，动量中性。
      4. ROC < 0： 当期收盘价低于 N 期前的收盘价，市场处于下降动量中。
8. DMI指标：Directional Movement Index，中文常译为趋向指标或方向移动指数。判断市场是否处于明确的趋势之中（上涨趋势或下跌趋势），并衡量该趋势的强弱程度。

   1. +DI（正向动向线，或称上升方向线）： 衡量当前上升趋势的强度。
   2. -DI（负向动向线，或称下降方向线）： 衡量当前下降趋势的强度。
   3. ADX（平均趋向指数）： 这是DMI系统的核心，它不指示趋势方向，只衡量趋势的强弱程度。
      1. +DM = 当前最高价 - 昨日最高价
      2. -DM = 昨日最低价 - 当前最低价
      3. TR = Max( (当前最高价 - 当前最低价), |当前最高价 - 昨日收盘价|, |当前最低价 - 昨日收盘价| )
      4. EMA平滑+DM，-DM，TR
      5. +DI = (+DM_smoothed / TR_smoothed) * 100
      6. (-DM_smoothed / TR_smoothed) * 100
      7. DX=|(+DI - -DI)| / |(+DI + -DI)| * 100
      8. ADX=EMA(DX, period)
9. Aroon：趋势是否开始、趋势强度以及趋势是否转换的动量指标。致力于在趋势的“黎明”初期就发现它

   1. Aroon Up = [(N - 自N周期内最高价以来的周期数) / N] * 100。
      1. 衡量自当前价格创下最近 N 期内的最高价以来，过去了多少期。量化了上升趋势的强度
   2. Aroon Down = [(N - 自N周期内最低价以来的周期数) / N] * 100
      1. 当前价格创下最近 N 期内的最低价以来，过去了多少期。量化了下降趋势的强度。
10. DBCD 中性策略：

    1. 没找到公认的版本。利用 DBCD 指标来生成交易信号，同时通过某种方式（如同时做多和做空）来对冲掉市场大盘的风险，从而追求稳定的、与市场涨跌无关的回报。

       1. SMA：当前数据相对于周期前的简单移动平均
       2. 计算与移动平均的偏离程度BIAS：(当前数据 - SMA) / SMA * 100
       3. 偏离差异: BIAS - (3周期前的BIAS)
       4. DBCD：（当前周期的BIAS - X周期前的BIAS） / X周期前的BIAS * 100
    2. DBCD 中性策略：

       1. 对整个市场生成DBCD指标，选取多头组合，信号最强的。选取空头组合，做空DBCD信号最弱的。
       2. “强组合”的涨幅必须超过“弱组合”的涨幅，就能盈利
    3. 策略

       1. 市场状态判断，当一半以上币种DBCD为负时清仓
       2. 每轮都只持有DBCD排名前三的币种
11. Dbcd-v2中性策略：

    1. 最后一步，计算DBCD 从简单移动平均SMA转为了EMA指数移动平均，带权重的，时间越近的权重越大
    2. 风控更严格：排名第一的币种DBCD转负就清仓
12. Dbcd-v3中性策略：

    1. 将成交量/成交额信息融入DBCD计算，没看到具体运用在哪了
    2. 风控：排名第一的币种DBCD转负就清仓
13. ACS和Demarker指标以及混合策略：

    1. ACS:
       1. ADX: 前面的一个参数
       2. ADX_close: 衡量市场趋势强度与当前价格关系的值： ADX/ 当日收盘价格
       3. ADX_close 标准差：衡量该值的波动性
    2. Demarker:超买超卖振荡器
       1. +DM = 当日最高价 - 昨日最高价
       2. -DM = 昨日最低价 - 当前最低价
       3. SMA 简单的移动平均+DM，-DM
       4. demarker: 平均后的+DM / (+DM + -DM)
    3. 策略：
       1. 买入条件：当Demarker指标值小于0.3（超卖）且ACS指标值小于0.0002（趋势稳定）时，执行卖出操作？
       2. 卖出条件：当Demarker指标值大于0.7（超买）且ACS指标值小于0.0002（趋势稳定）时，执行买入操作？
